Beth's AI Tools

This repo contains a list of the tools I've made to make it easy to interact with OpenAI's artificial intelligences, such as GPT-3 and DALL-E.

Since the OpenAI APIs cost money, you will need to provide your own API token in order to use these tools.

How to use

To get started with these tools:

  1. Create an account at beta.openai.com/signup.
  2. Generate an API key and copy it to your clipboard.
  3. Go to one of the tools below, and paste your API key where it asks for an API token.
  4. Type in your prompt and submit it.
  5. Wait for the result!

At the time of writing this, you get $18 worth of free credits when you first sign up for an OpenAI account.

Note: Your token is stored locally in your web browser. It is never transmitted to me in any capacity.

All of these tools were built for function, not beauty.

GPT-3 Chat

Description

This tool allows you to send prompts to GPT-3, and it displays the results in the format of a text chat.

About this tool

This was the first AI tool I built. After I first tried out GPT-3, I got so addicted to it, that I started to wish I could do prompts from my phone. I decided to build this web interface so I could do it from either my phone or desktop, and I thought the chat format was fun and immersive.

Its creation was very rushed and I didn't work on it much after it reached a functional state, but it served as the base on which I built the other two AI tools.

DALL-E Web Interface

Description

This tool lets you type in a prompt, and DALL-E generates an image based on your prompt. You can choose from three different resolutions, and you can also generate up to 10 images at a time.

This tool doesn't save your images for you, so make sure you save them by right clicking and selecting Save As.

About this tool

I've been obsessed with AI-generated images ever since the technology became even remotely accessible. When the DALL-E API left became open to the public, I had to start playing with it right away.

Like with GPT-3, it was only a matter of time before I HAD to have an easy way to do it from my phone or desktop, and I'm really happy with how this tool turned out.
